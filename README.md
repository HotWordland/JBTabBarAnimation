## Requirements

- iOS 11.0+
- Xcode 10.0+

## Installation

JBTabBarAnimation is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'JBTabBarAnimation'
```
